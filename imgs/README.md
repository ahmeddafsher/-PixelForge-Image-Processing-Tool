## 📌 Overview

**Image Color Converter** is a simple but powerful image processing tool built with **Python** and **OpenCV**.

The project allows users to load images and apply different color transformations through a simple command-line interface.

This project was created to practice:
- 🐍 Python programming
- 👁️ Computer Vision basics
- 🎨 Image processing using OpenCV

---

## ✨ Features

✅ Load custom images (PNG / JPG)  
✅ Display original images  
✅ Convert images to grayscale  
✅ Convert images to YCrCb color space  
✅ Convert images to Lab color space  
✅ Interactive menu system  
✅ Add your own images easily  
✅ Exit program anytime using `q`
